About The Page

Noir's Headers and Footers is a project for Seatwork 3 in my Web Design course, aimed at enhancing web pages with engaging header and footer animations. The goal is to create dynamic and cool effects that make these sections stand out and improve user interaction. This involves implementing various animations such as fade-ins, hover transitions, and slide-up effects to draw attention and provide a pleasant visual experience. The project showcases creativity and technical skills in web design.
